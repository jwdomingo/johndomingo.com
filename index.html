The following page is a hand-coded website chronicling my journey to becoming a web developer. Please note the 'About' and 'Blog' buttons are toggled using a simple CSS hack using the :target pseudo class (no JavaScript yet!) A little tacky, I realize but once I learn jQuery I will replace it!
